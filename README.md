# Telegram-бот записи (маникюр) — aiogram + SQLite

Готовый бот для мастера по маникюру:

- запись на свободные слоты через **inline-кнопки**
- календарь на **30 дней вперёд**
- один пользователь — **только 1 активная запись**
- **SQLite** база
- админ-панель (по `ADMIN_ID`): рабочие дни, слоты, закрытие дня, отмена записей, просмотр расписания
- обязательная **проверка подписки** на канал
- кнопки **«Прайсы»** и **«Портфолио»**
- автонапоминание за 24 часа через **APScheduler**, восстановление задач после перезапуска

## Установка

1) Перейдите в папку проекта:

```bash
cd manicure_bot
```

2) Создайте виртуальное окружение и активируйте:

Windows (PowerShell):

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
```

3) Установите зависимости:

```bash
pip install -r requirements.txt
```

4) Создайте файл `.env` рядом с `bot.py` (можно скопировать из `.env.example`) и заполните:

- `BOT_TOKEN`
- `ADMIN_ID`
- `CHANNEL_ID`, `CHANNEL_LINK`
- (опционально) `SCHEDULE_CHANNEL_ID`

## Запуск

```bash
python bot.py
```

## Важно

- Для `CHANNEL_ID` используйте **ID канала** вида `-100...`
- Бот должен иметь право читать участников канала (обычно достаточно быть админом канала).
